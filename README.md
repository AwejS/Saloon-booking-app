# Saloon-booking-app
A Simple mobile app for booking service at saloons in React Native 

### Features
#### 1.User login using Async reactNative storage
#### 2.User can select slot at listed saloons 
#### 3.Booking form with time,date and service with price options also email before submit
#### 4.Updating time slots if booked previously with notification like "Not available Slot"
#### 5.Appointment Booking page to see all booking at one place also can cancel appointment
